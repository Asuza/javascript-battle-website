Rules


The first rule of Javascript Battle is...there are no rules! Ok, really, there are some rules, but they are totally reasonable.

  1. Create a GitHub account, if you don't have one.
  2. Go to our readme and follow all the steps to get started.
  3. Learn some Javascript, write some code, make your hero awesome, and watch the results.
